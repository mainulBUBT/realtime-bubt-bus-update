# Plan: Fix FCM Notifications in Student App

## Problem
FCM notifications are not showing in the Vue.js + Capacitor student app.

## Root Causes Identified

### 1. Missing `google-services.json`
The Firebase configuration file is completely missing from the project. This is **critical** - without it, FCM cannot work.

### 2. Missing Firebase Messaging Service in AndroidManifest.xml
The AndroidManifest.xml (`frontend/capacitor-student/app/src/main/AndroidManifest.xml:27-45`) is missing the Firebase Messaging service declaration:
```xml
<service
    android:name="com.google.firebase.messaging.CapacitorFirebaseMessagingService"
    android:exported="false" />
```

### 3. Missing Android 13+ Notification Permission
Android 13+ requires `android.permission.POST_NOTIFICATIONS`. While it's declared, the app doesn't request it at runtime.

### 4. Firebase Messaging Not Initialized
The composable `useFirebaseMessaging.js` exists but is **never called** anywhere in the app. FCM must be initialized on app startup.

### 5. Missing Capacitor Config File
The `capacitor.config.js` is in `scripts/capacitor-student.config.js` but not copied to `capacitor-student/`.

---

## Solution Steps

### Step 1: Add google-services.json
- Obtain `google-services.json` from Firebase Console
- Place it at: `frontend/capacitor-student/google-services.json`
- Ensure package name matches `appId` in capacitor config: `com.bustracker.student`

### Step 2: Add Firebase Messaging Service to AndroidManifest.xml
Add the service declaration after the MainActivity:
```xml
<service
    android:name="com.google.firebase.messaging.CapacitorFirebaseMessagingService"
    android:exported="false" />
```

### Step 3: Add Meta-data for Auto-init
Add in application tag:
```xml
<meta-data
    android:name="com.google.firebase.messaging.auto_init_enabled"
    android:value="false" />
<meta-data
    android:name="firebase_messaging_auto_init_enabled"
    android:value="false" />
```

### Step 4: Initialize Firebase Messaging in App
In `frontend/src/App.vue` onMounted, call the useFirebaseMessaging composable:
```js
import { useFirebaseMessaging } from '@/composables/useFirebaseMessaging'

const firebaseMessaging = useFirebaseMessaging()
firebaseMessaging.initialize({
  topic: 'student-notifications',
  onMessage: (notification) => {
    console.log('FCM Message:', notification)
  }
})
```

### Step 5: Copy capacitor.config.js
Copy `scripts/capacitor-student.config.js` to `capacitor-student/capacitor.config.js`

### Step 6: Rebuild APK
```bash
cd frontend
npm run apk:student:prod
```
